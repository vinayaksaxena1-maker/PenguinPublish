# PenguinPublish

A premium publishing company website built with **React + Vite + TypeScript**.

## Features

- Modern, responsive UI with glassmorphism design
- Featured books showcase
- Testimonials section
- Tailwind CSS styling

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
