# MB Publisher Website Rules

## Tech Stack
- Use React + TypeScript + Tailwind CSS only.
- Maintain clean and reusable component architecture.

## Design System
- Follow Home Page design language across all pages.
- Use premium white + orange theme.
- Maintain same typography throughout website.
- Maintain same spacing system throughout website.
- Maintain same container width throughout website.
- Maintain same hover effects and animations throughout website.

## Safety Rules
- Never delete existing sections without permission.
- Never modify Header without permission.
- Never modify Footer without permission.
- Never break existing functionality.
- Preserve SEO structure.

## Mobile Responsiveness
- Make all pages fully responsive.
- Do not disturb desktop layout while fixing mobile layout.
- Tablet responsiveness is required.

## Workflow
- First review existing code.
- Explain what changes will be made.
- Generate code only after analysis.
- Wait for approval before major structural changes.

## Content Rules
- Never use Lorem Ipsum.
- Use professional publishing industry content.
- Keep content client-ready.

## MB Publisher Identity
- MB Publisher is a premium book publishing and printing company.
- Design should feel modern, trustworthy, premium and conversion-focused.
